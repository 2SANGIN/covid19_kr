# 국내 코로나 현황 컴포넌트

국내 코로나 현황 컴포넌트입니다.
보건복지부 및 각 지자체 홈페이지에서 현황을 크롤링하여 센서로 표시합니다.

통합구성요소를 지원하며, 시/도별 현황 및 도시별 현황을 각각 표시할 수 있습니다.
